# 项目说明

## 支持功能

* 甲醛（HCHO）浓度检测
* 二氧化碳（CO2）浓度检测
* 一氧化碳（CO）浓度检测
* 臭氧（O3）浓度检测
* 一氧化硫（SO）浓度检测
* 硫化氢（HS）浓度检测
* PM1.0、PM2.5、PM10浓度检测
* TVOC浓度检测
* 温度湿度检测

## 传感器选型

> 选型原则，不计成本，选择民用产品中性能最好的传感器。

甲醛传感：英国Dart WZ-H3-N

二氧化碳传感器：瑞典SenseAir S88-LP

一氧化碳传感器：英国Dart
